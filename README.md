# Build the autograd engine

**Neural Networks: Zero to Hero - Module 1 - Project 1**
Companion lecture: [The spelled-out intro to neural networks and backpropagation](https://youtu.be/VMj-3S1tku0) by Andrej Karpathy.

Learn how neural networks *actually* learn - by building a tiny automatic-differentiation
engine from scratch, the same idea behind PyTorch. Instead of just watching the lecture, you'll
build it yourself, with an AI tutor guiding you one step at a time.

## What you need
- Python 3.9+ and `pytest` (`pip install pytest`)
- Any coding agent or AI assistant that can read the project files and run shell commands
  (Claude, Codex, Cursor, or another tool)

## How to start
1. Download or clone this project and `cd` into it.
2. Open it with your preferred coding agent.
3. Just say hi. Your tutor takes it from there.

For tool-specific setup notes, see [SETUP.md](./SETUP.md). It covers Claude Code, Codex,
Cursor, and generic coding assistants.

You'll build a real, trainable neuron from scratch in about an hour. Your code goes in
`engine.py`. Ask for a hint when stuck, ask the tutor to check your work to run the tests,
and ask to move on when ready for the next milestone.

The tests decide when you're done - not the tutor. When they all pass, you've built it.

Next up: **Project 2 - Build & train a neural net**, where you put this engine to work and
train a real (tiny) network.
